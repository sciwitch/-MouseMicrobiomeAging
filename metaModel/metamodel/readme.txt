Contains scripts and data relating to the reconstruction and analysis of the metabolic metamodel.
Top-level folder for metabolic metaorganism modeling results.

The required data to run this metamodel with these scripts has been seperately uploaded to a zenodo record (doi.org/10.5281/zenodo.10844503)
and can be obtained from https://zenodo.org/records/10844503/files/metamodel_analysis.zip?download=1


scripts
-------
Code used in the analyses.

results
-------
Contains results.

models
------
Contains models.

processedData
-------------
Contains preprocessed data for modelling.

rawData
-------
Contains raw data.

mapping
-------
Mapping between different IDs as well as between human and mouse genes.
